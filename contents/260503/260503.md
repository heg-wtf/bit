## abyss

큰 줄기는 두 가지. Claude Code 통합 Phase 1~6 마무리, 그리고 대시보드 강화.

CC changelog plan을 6단계로 쪼개서 진행하던 게 이번 주에 끝났다. env var 주입(Phase 1), MCP alwaysLoad와 500K meta(Phase 2), PreCompact 훅(Phase 3), PostToolUse telemetry와 skill-conditional 훅(Phase 4), disableSkillShellExecution과 sandbox.deniedDomains(Phase 5), ultrareview wrapper와 per-bot effort level(Phase 6). 이걸 다 끝내고 plan 문서들도 싹 지웠다. 끝난 plan은 빨리 치우는 게 정신건강에 좋다.

대시보드도 많이 건드렸다. 빈도 히트맵 추가하고, Telegram 봇 프로필 사진 보여주고, 채팅 UI를 양방향 동기화로 한번 올렸다가 되돌리고 다시 top-level `/chat`으로 분리해서 올렸다. 파일 첨부(이미지·PDF)도 붙였고, raw next build 로그 대신 빌드 체크리스트로 바꿨다. 한 번에 들어가니 정리가 깔끔해졌다.

그 외 자잘한 것들. openai_compat 백엔드 추가해서 MiniMax 직접 붙였고, bot.yaml에서 api_key 바로 받게 했다. 마크다운→텔레그램 HTML 변환에서 링크 URL 새니타이징하는 보안 픽스도 했다. 회귀 테스트로 display_name 복원이랑 사이드바 collapse 잠금 잡고, 커버리지도 끌어올렸다. 버전은 2026.04.29 → 2026.04.30로 두 번 올렸다.

조용한 주가 몇 번 지나면 이렇게 한 번에 터진다. 다음 주는 좀 쉬어가도 될 듯.
