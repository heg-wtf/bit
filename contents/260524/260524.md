이번 주도 abyss 몰빵. PR 10개. 큰 줄기는 "공진화(co-evolution)"다.

## abyss

봇이랑 사용자가 같이 자라는 그림을 그리고 있다. Phase 1로 숫자 피드백(1/2/3)을 받기 시작했고, Phase 2a에선 ABOUT_ME라는 공유 지식 베이스를 깔았다. 봇이 사용자에 대해 아는 걸 한 곳에 모으는 거다. Phase 2b에선 AI가 ABOUT_ME를 직접 제안하고 사용자가 확인하는 흐름까지 붙였다. 대시보드에서 바로 본다. 이 흐름이 만들어지니까 봇이 "기억하는 느낌"이 확 살아난다.

문서도 한 번 싹 정리했다. ABOUT_ME랑 숫자 피드백을 README, ARCHITECTURE, TECHNICAL-NOTES, ROADMAP, CLAUDE.md에 한 번에 박았다. 새 개념 들어올 때 문서 안 고치면 나중에 후회한다.

자잘한 것들도 많았다. 루틴 답장 푸시 알림, 루틴 상세 자동 스크롤 픽스, 봇 별명(role label) — "앤 (집사)" 같은 — 옵션 추가, 채팅·루틴 읽음/안읽음 표시. about-me 페이지 영문 카피 번역도 했고, 마지막으로 Python 3.12 지원하느라 PEP 758 문법 걷어냈다. 의존성 환경 맞추는 건 어쩔 수 없다.

공진화 그림이 슬슬 형태가 잡힌다. 다음 Phase가 기대된다.
